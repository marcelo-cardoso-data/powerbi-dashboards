### Analitico_Vendas_Globais/
- `Analitico_Vendas_Globais.pbix` – Dashboard de indicadores de **vendas globais fictícias**  
- `base_dados_Analitico_Vendas_Globais.csv` – Base de dados utilizada para gerar o painel 

**Descrição do Analitico_Vendas_Globais:**
- Média de Desconto por SubCategoria  
- Total de Vendas por Categoria  
- Total de Vendas Global  
- Média dos Valores de Vnendas por País
- Total de Vendas Por País e Prioridade
